运行代码

  1. 训练

    将相应分辨率图像文件夹改为image，输入以下命令进行训练神经网络。

    python train_lsgan.py

    ​

  2. 测试

    利用已经训练好的模型随机产生更多图像。

    python test.py

    ​

  3. 修复

    将待修复图像放入uncompletion_image文件夹，利用生成好的模型修复。

    python train_completion.py
