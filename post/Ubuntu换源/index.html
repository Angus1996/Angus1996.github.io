Ubuntu换源

2018-03-27

Ubuntu命令行查看系统版本号

sudo lsb_release -a

第一步：备份原来的源文件

cd /etc/apt

然后会显示下面的源文件sources.list
输入命令
sudo cp sources.list sources.list.bak
就是将sources.list备份到sources.list.bak

第二部：替换源

输入sudo vim sources.list

进入VIM编辑器，输入ggdG删除所有内容，再粘贴以下的源

按下ESC，输入:wq保存退出
