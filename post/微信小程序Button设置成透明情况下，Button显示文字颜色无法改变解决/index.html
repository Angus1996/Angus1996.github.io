微信小程序Button设置成透明情况下，文字颜色无法改变解决方法

微信小程序的button组件，当将其属性“plain”设置成 true 时，表示将改Button组件显示为镂空，背景色透明。在这种情况下，如果我们通过 .wxss文件将button的文字改变颜色，是无法达到目的的。

在搜索参考 https://www.cnblogs.com/PeunZhang/p/4089894.html 下，借鉴CSS实现背景透明，文字不透明的方法，成功应用在微信小程序的button中。

css3的rgba(red, green, blue, alpha)，alpha的取值从 0 到 1，表示不透明度，如rgba(255,255,255,0.8)。

具体 .wxss部分代码如下，将button设置成不透明度50%，文字颜色为白色：

.btn1{
    background-color:rgba(0,0,0,0.5);
    color: white;
}
