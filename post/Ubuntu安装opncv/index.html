Ubuntu安装opncv

2018-03-01 1 min read # opencv # Linux

今天跑程序做图像修复的时候遇到下面的问题：

import cv2
ImportError:No module named cv2

发现Ubuntu里面没有安装OpenCV视觉库。安装方法如下：

下载opencv-3.0.0.zip

安装包在我的百度网盘。链接: https://pan.baidu.com/s/1daP0mi 密码: kav1

解压并依次输入以下代码

在下载目录打开终端，输入以下命令

unzip opencv-3.0.0.zip
cd opencv-3.0.0
mkdir build
cd build
cmake ..

安装OpenCv

sudo apt-get install python-opencv

注意：安装命令中全部为小写。
