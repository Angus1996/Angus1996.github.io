python3获取当前路径的父路径

# 获取当前路径
pwd = os.getcwd()
#当前文件的父路径
father_path = os.path.abspath(os.path.dirname(pwd) + os.path.sep+".")
#当前文件的前两级目录
grader_father = os.path.abspath(os.path.dirname(pwd) + os.path.sep + "..")

上述代码可用于：获取父路径和切换到新的父路径下的其他目录再连接其他目录下的文件
